icCube Schema & Cubes

This directory contains schemas & cubes shipped with icCube.

Upon the first startup of icCube, those files are copied within the cubes directory
of the application directory; on Windows XP this directory is:

C:\Documents and Settings\icCubeUser\.icCube\cubes

Open the 'Monitoring' console to get the exact location of this directory and its actual
content: http://localhost:8282/icCube.html
